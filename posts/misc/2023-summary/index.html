2023 年度小總結與 2024 年度新目標

2023 也就要過去了，迎來嶄新的 2024，在今年的最後一天，替自己做個小總結。

2023

第一點想說的是，「至少一個月要生出一篇文章」，但變成一個月真的只剩一篇文章。
大部分都是到月底才要寫，可能因為工作與休閒活動的關係，寫文章的優先權並沒有那麼高。
但至少當初自己的要求有達成，除了我出意外的那兩個月修復期。

首先是今年三月，遇到了 14 天教召，非常的刺激與痛苦，每天全副武裝。
行軍與打靶也免不了，而且還多了戰術演練，讓你在街道與沙灘跑跑跳跳。
最後還預告 2025 年，我們再相見，真的是受不了耶…

今年三月家人跌倒，膝蓋骨折。四月自己出車禍，左手骨折。
那段日子很不好受，不過親人、朋友與同事們的關心有讓自己好受一點，醫生也說恢復得蠻快的。
大概是六月時，醫生說沒問題可以正常生活了。
明年要把這個鋼板拆下來，想問醫生能不能帶回家，畢竟這好幾萬…

七月初不知道哪根筋不對，報名了八月的多益測驗。可能出車禍讓自己腦子怪怪的。
兩個月的時間，白天要上班，傍晚要通勤，晚上到家還要玩遊戲
從來沒有考過多益，並且我自認為我英文超爛…後來考出來好像還可以啦… 5xx（爛死）

考多益之前還收到面試，也就是 About page 內的那家光電產業。
其實我原先只是抱持著沒上也無所謂的心態去，我承認我是去參觀，投影片隨便做，面試都在聊天完全不緊張。
而且一面跟二面只差兩天，我記得第二天 9:30 面試，從宜蘭殺去新竹，超怕遲到…

不過我在這邊的文章有派上用場，當場直接開來給面試官，也就是我現在的主管看。
寫到這想起以前的恩師說過：

Info
平時要慢慢累積一些東西，一來你臨時要用可以拿出來用，二來當這些東西累積夠了，它是會爆發出來並帶來回饋的。

所以我常常後悔沒有累積英文。

10/13 離職，10/16 上班。
不給自己鬆懈時間，就是怕自己鬆懈下來，新的工作職務繁多。
大部分是電信與網路，另外還有內部系統開發，開發的部分我可以天馬行空自由創作。
電信與網路的部分，希望自己可以過試用期考核…拿 10 年前學的網路技能來挑戰現代網路是不是有點自不量力…

今年學到的技能，比較 ok 的大概就 Git, Lua 與 Vue.js。
Git 以前我就只為 add, commit, push ，這邊會提到主要是體會多人合作時會發生什麼情形，像是 Force 不小心蓋掉整個 branch 的狀況，是有遇過但不是我弄的。
Lua 是為了介接產品的使用者使用的網路頁面與內部用 Language C 開發的系統，剛開始用起來頭其實蠻痛的，有些邏輯很特別 … 但再痛也不會比 Language C 痛啦。
Vue 是因為我得開發公司用的內部系統，想說給自己一個挑戰，原先是想用 react 但我學沒多久就放棄了，因為這個 blog 剛開始用 vuepress，那時候覺得 vue 比較簡單一些。
剛好可以趁這個時候再學得更深入一些，像這陣子就都在學 vue router 做 Single Page Application，其實還蠻有趣，但大部分時間都是 route 轉不過去。

2024

2024 是新的一年。先替自己訂一些目標，有訂可能不會做，沒訂肯定就不會做。

  • 保持一個月一篇文章的節奏，如果允許可以一個月兩篇。
  • 把我最熟悉的 Language: Java 好好的 Deep Learning。
  • 把 Vue 學到一個水準。
  • 考 1~2 張 Cloud 的證照。預計明年年初報名，今年年底有開始學習，但是折價券不能用在明年，希望明年年初可以盡快拿到折價券…
  • 考 CCNP。同事是 CCIE 我還不問爆他。
  • 學習 Docker, K8s。大概學個基本操作就好…
  • 英文想再進步一點。

其他目標我就不列在這了…像是要去屏東玩之類的，跟這個 blog 沒有關係。
也許會列在別的地方，也希望各位的新年新目標能夠達成。
