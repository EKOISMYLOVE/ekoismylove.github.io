2024 也就要過去了，迎來嶄新的 2025，在今年的最後一天，替自己做個小總結。

2024

與去年一樣，第一點想說的是，「至少一個月要生出一篇文章」，但今年變成下半年都沒寫。
坦白說連這篇文章都是搾時間出來寫的…

Q2 時跟鬼一樣強大的同事離職，很多事情要硬吃下來。

Q3 一個防火牆上線的專案搞得自己很狼狽，手上的資料交接到我這邊已經是第三手，很多資訊已與之前的不一致，但好加在 Q3 有做完。
雖然明年這個防火牆又要改東改西的…

Q4 安排了外部訓練以及拆鋼板，整整少了五個禮拜可以做事。 但答應人的 Deadline 就是要遵守，變成常常要搞到七晚八晚的 另外還加了一個海外辦公室…一來一往就是12個小時。

老闆說要把我訓練成扛壩子，看來不是在忽悠我…

年初有去考了 AWS 證照，雖然是最最最基本的，後來工作好像也沒機會使用，但就當作學習吧，希望明年能再接再厲，畢竟考過有給 50% off 折價券。

Q1, Q2 還沈浸在 Vue 跟 PHP 中，Q3 就要硬著頭皮管網路了…都在學習複習 Cisco、Fortinet 設備如何設定…還要學 PBX、thinclient…外加一個海外辦公室要管。

Q4 開始重拾學很久的 Java，應該說是學習 Spring Cloud 與複習 Spring MVC。 希望明年能把 Spring Cloud 一起學起來，再加深自己的雲端技術。

喔對，明年我查過了，我好像不用教育招集！

今年學到的事情不少…

  1. 年初還是新人、年底就變老人…
  • Q3 補了一位新工程師，因為他是新鮮人沒有相關工作經驗那種，很多基本功要慢慢的邊做邊教他。
  • 平常不會問我問題的人會跑來問我（寫這篇文章的時候又多一個），一定不是因為我夠強，應該是因為只能問我了…
  1. 拋給別人處理的事情，你不相信他會做完，你就要一直盯著他…
  • 我不是主管職，我不太喜歡請人做事情，100％丟給他那種請人做。
  • 但我有個海外辦公室要管…
  • 溝通很重要、要知道如何麻煩別人做事…
  1. 技術
  • TroubleShooting
  • Vue
  • PHP、Perl
  • LAN Switching
  • Firewall(Firepower, Fortigate)
  • 溝通
  • 好好溝通
  • 跟老闆好好溝通

Review 2024

2024 是新的一年。先替自己訂一些目標，有訂可能不會做，沒訂肯定就不會做。

回來看我去年許了哪些鬼願望:

  • 保持一個月一篇文章的節奏，如果允許可以一個月兩篇。

沒做到：下半年沒有，但有學其他東西…

  • 把我最熟悉的 Language: Java 好好的 Deep Learning。

沒做到：不用解釋。

  • 把 Vue 學到一個水準。

有做到：專案用很多，基本上基礎的生命週期、Router、資料傳遞都有了解到，就是怕自己金魚腦忘記…

  • 考 1~2 張 Cloud 的證照。預計明年年初報名，今年年底有開始學習，但是折價券不能用在明年，希望明年年初可以盡快拿到折價券…

有做到。

  • 考 CCNP。同事是 CCIE 我還不問爆他。

沒做到：排到明年才要考…

  • 學習 Docker, K8s。大概學個基本操作就好…

沒做到：金魚腦忘記了。

  • 英文想再進步一點。

不確定：我覺得有啦…

其他目標我就不列在這了…像是要去美國玩之類的，跟這個 blog 沒有關係。
也許會列在別的地方，也希望各位的新年新目標能夠達成。

2025

還是來許個願吧 :

必修：

  • 學習：Spring Cloud、Docker、K8s
  • 加強：Cisco Network、Fortinet 鐵三角
  • 證照：兩張

選修：

  • 學習：React.js
  • 加強：Debian(好幾次更新，有些預設網路指令都改了)、自動化技術、監控
  • 證照：一張
